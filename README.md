# Traveling Salesman Problem (TSP) Solver

CMSC142 ST-1L 2025-2026 2nd Semester
Chan | Tamayo


A C-based implementation exploring different algorithmic solutions to the Traveling Salesman Problem (TSP). 
This project was developed to demonstrate the trade-offs between exact computational methods and heuristic approximations.


## How To Use: 
Build the file using the make file
- make tsp

Run the file
- ./tsp_solver
